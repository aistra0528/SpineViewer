# Spine Viewer

Spine viewer for Android.

Supported Spine version: 4.0 (Can be configured in [gradle.properties](gradle.properties))

## How to use

Tap the left edge of the screen: Import files (`.atlas` + `.png` + `.json`/`.skel`) or background (`.jpg`/`.png`).

Tap: Switch between skins

Long press: Switch between animations

Touch & Move: Pan view

Touch & Scale: Zoom view
